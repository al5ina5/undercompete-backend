# UNDER Compete

### Setup 

<br>

##### Clone the repo
```
$ git clone https://github.com/al5ina5/undercompete-backend
```
##### Install dependencies
```
$ npm install / yarn save
```

##### Create a .env file
```
$ touch .env
```

> Sample .env
```
DB_URL = "mongodb://localhost:27017/undercompeteDB"
```

##### Start the server
```
$ npm start / yarn start
```
